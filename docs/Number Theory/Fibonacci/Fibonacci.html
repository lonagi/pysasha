Числа Фибоначчи — элементы числовой последовательности, в которой первые два числа равны либо 1 и 1, либо 0 и 1, а каждое последующее число равно сумме двух предыдущих чисел.

Более формально, последовательность чисел Фибоначчи $ \{F_n\} $ задаётся линейным рекуррентным соотношением:

$ F_0=0,\; F_1=1,\;\;\; F_n={F_{n-1}}+{F_{n-2}},\;\;\; n \geq2,\; n \in Z $

А теперь смотрите. Новый алгоритм.
Есть такая формула Бине: $ {F_n}={ {\varphi^n - (-\varphi)^{-n}}\over{2\varphi-1} } $, где $\varphi$ - золотое сечение, о котором кстати мы потом ещё поговорим, в следующих статьях.

Единстенный недостаток формулы в том, что она выдаёт не целое число. Придётся округлять.

А ещё кстати если подавать на вход большие индексы, то будут проблемы на выходе.
Поэтому воспользуемся некоторыми функциями из Sympy, чтобы избежать этих проблем.

#roundi - если True, то число будет компактным, и это позволит ускорить нахождение числа
def Binetformula(index,roundi=True):
    from sympy import Pow as mmmpow
    from sympy import Integer as mmmint
    from sympy import N as mmmn
    b = (mmmpow(1.6185,index)-mmmpow(-0.6185,index))/2.237
    if(not roundi):
        return mmmint(b)
    else:
        return mmmn(b)

#Здесь вручную происходит матричное возведение в степень

def __pow(x,n,I):
    if n==0:
        return I
    elif n==1:
        return x
    else:
        y = __pow(x,n//2,I)
        y = y.dot(y)
        if n%2:
            y = x.dot(y)
        return y

def MatrixFibonacci(index):
    F = __pow(np.array(((1,1),(1,0)),dtype=object),index,np.eye(2))
    return F[0][1]

def Divisors(num): 
    from math import sqrt as mmsq
    s=set([1])
    i=1
    a=int(mmsq(num)+1)
    while i<=a: 
        if(num//i==num):
            i+=1
            continue
        if (num%i==0): 
            if (num//i!=i): 
                s.add(num//i)
            s.add(i)
        i+=1
    return s

    from math import log as mlog

У нас математика, а тут бац, и слова!

Но не всё так просто. Давайте разберём, что такое слова Фибоначчи.

Слово Фибоначчи формируется путём повторения конкатенации тем же образом, что и числа Фибоначчи образуются путём повторяемых сложений.

То есть, если мы имеем начально 0,1
Следующее число Фибоначи 0+1=1, получится ряд 0,1,1
Снова прибавляем два последних, 1+1=2 и будет ряд 0,1,1,2
Но мы только что пошли по "ветке", когда складываем числа при помощи арифметики.
А нам же нужна конкатенация. Это когда мы "склеиваем" элементы

Например, имеем начальный ряд 0,1
"Склеиваем" их..получаем 01; добавим в ряд и получим 0,1,01
Но есть два НО.
Во 1-ых, ряд слова Фибоначчи начинается с $ F_0=0 $ и $ F_1=01 $
Во 2-ых, лучше записывать ряд без запятых (это же слово как никак)
А если всё же и перечислять через запятую, то разделять число по цифрам, например 01 будет 0,1

А теперь напишем программу, которая будет находить нам N-ый элемент слова Фибоначчи, если мы подадим на вход индекс

def IterationFibonacciWord(index):
    Sn_1 = "0"
    Sn = "01"
    tmp = "" 
    for i in range(2, index + 1): 
        tmp = Sn
        Sn += Sn_1 
        Sn_1 = tmp 
    return Sn
